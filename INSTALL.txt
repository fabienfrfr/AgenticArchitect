# AgenticArchitect - Installation Guide

## Prerequisites
- Docker + Docker Compose
- NVIDIA GPU (for Nemotron-3)
- Python 3.10+

## Setup
1. Clone this repository
2. Run the bootstrap script:
   ```bash
   python bootstrap.py
   ```
3. Start the services:
   ```bash
   docker-compose -f infra/docker-compose.yml up --build
   ```
4. Access the dashboard at [http://localhost:3000](http://localhost:3000)

## Testing
Run the test suite:
```bash
docker-compose -f infra/docker-compose.yml exec architect pytest tests/
```
